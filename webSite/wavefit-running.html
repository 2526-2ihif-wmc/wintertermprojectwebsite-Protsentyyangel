Running Guide

Start your running journey today

Benefits of Running

  • Improves Heart Health: Strengthens your heart and circulation
  • Burns Calories: Great for weight loss
  • Builds Endurance: Increases stamina
  • Mental Health: Reduces stress and improves mood
  • Strengthens Bones: Builds bone density
  • Better Sleep: Helps you sleep deeper

Types of Running

Easy Run

Comfortable pace where you can talk. Most runs should be easy.

Tempo Run

Comfortably hard pace. You can speak short sentences but not full conversations.

Interval Training

Alternate between fast and slow. Example: 5x (3min fast, 2min slow).

Long Run

Longer distance at easy pace. Builds endurance. Once per week.

Type How Often Benefits
Easy Run 3-4 times/week Builds base fitness
Tempo Run 1 time/week Improves speed
Intervals 1 time/week Increases speed
Long Run 1 time/week Builds endurance

Proper Running Form

  • Posture: Body upright with slight forward lean
  • Head: Look ahead, not down
  • Shoulders: Relaxed and down
  • Arms: Bent at 90 degrees, swing naturally
  • Hands: Relaxed, not tight fists
  • Feet: Land on midfoot, not heels
  • Steps: Quick and light, 170-180 per minute

Mistakes to Avoid

  • Running Too Fast: Most beginners run easy runs too fast
  • Long Steps: Don't overstride
  • No Warm-up: Always walk or jog slowly for 5-10 minutes first
  • Ignoring Pain: Stop if you feel sharp pain
  • Wrong Shoes: Get proper running shoes
  • Too Much Too Soon: Don't increase distance more than 10% per week
